Smart ATS — Applicant Tracking System

A complete recruitment workflow system built with Django & MySQL. Featuring resume parsing, automated scoring, secure HR invites, and role-based dashboards used in real ATS platforms.

Browse Jobs HR / Admin Login

Core Capabilities

Job Portal

Public job listings with quick apply, no login required.

Resume Parsing

Extracts skills, experience, projects & contact info from PDFs.

Match Scoring

Automatic scoring based on skills, experience & JD keywords.

HR Dashboard

Create & edit jobs, review applicants, track status.

Admin Control

Manage all HRs, jobs & applications with full oversight.

Secure Uploads

Strict PDF-only, 5MB limit, sanitized storage & safe parsing.

Role-Based Access Hierarchy

Superadmin

System-level control. Creates Admin accounts via Django Admin.

Admin

Manages HR users, oversees jobs & applications.

HR Recruiter

Posts jobs, screens candidates, updates status.

Candidate

Applies directly from the job portal without signup.

User Creation Workflow

Superadmin → Admin

Admins cannot self-register. Only Superadmin can create them.

→

Admin → HR (Invite System)

HR accounts are created only via secure invite links with tokens.
